# Plant Pending v1.0 official release

## Final polish

- Right rail panels now use the full flyout space instead of small capped lists.
- Zones, groups, legend, and debug panels scroll naturally inside the inspector.
- Fixed the zone settings close behavior so text selection inside the modal does not accidentally close it.
- Kept the app branded as Plant Pending v 1.0.

## Notes

Use the normal dev commands:

```powershell
npm install --registry=https://registry.npmjs.org
npm run dev
```

If dependencies are already installed in the same folder, use:

```powershell
npm run dev
```
