# Plant Pending v1.0.15 – Soften plant card name overlay

The v1.0.14 full-width black name bar was too heavy and covered too much of the image.

Changed:
- Replaced the full-width name bar with a smaller lower-left translucent name pill.
- Reduced opacity from heavy black to a softer transparent black.
- Reduced name font size.
- Replaced the large bottom gradient with a smaller corner gradient.
- Keeps the plant name readable while letting the image show through.

Build check:
- npm run build passed.
