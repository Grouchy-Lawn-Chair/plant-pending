# Plant Pending v1.0.3 — Welcome, example plan, and help center

## Changed

- Startup canvas now uses the white-background Plant Pending logo.
- Empty canvas callout now says: upload a top-down image, load an example plan, or start drawing zones.
- “Upload a top-down image” is clickable and opens image upload.
- “Load an example plan” is clickable and loads the bundled example plan.
- Added bundled example plan at `public/examples/example-plan.json`.
- Added first-time welcome guide with five workflow steps.
- Added “Show this when the app opens” checkbox.
- Added `? Help` button in the header.
- Added full-height right-side Help Center panel with search and sections.

## Tested

- `npm run typecheck`
- `npm run build`
