# Garden Planner v109 – Card sizing + inspector workflow

## Changes

- Plant card metric squares capped to 50px high.
- Plant card is smaller overall.
- Price tile no longer has an icon or label, just the price text.
- Water icon cleaned up.
- Pollinator tile uses a bee icon.
- Zone Settings modal can be closed by clicking outside it.
- Zone Settings modal can be dragged by the header.
- Inspector is now split into tabs: Zones, Groups, Debug.
- Save/load/import/export/new/print moved into a File menu in the top header.
- Header version updated to Garden Planner ver. 109.
