# Plant Pending v1.0.2 save/open fix

## Fixes

- Added File > Open / load plan... so all saved plans are accessible, not just a short recent list.
- Recent plans are now sorted by most recently updated.
- Saving with the same plan name updates that saved plan instead of creating a hidden duplicate with a new ID.
- Saved plan list shows plant count, zone count, and updated time.
- Saved plan modal supports delete.

## Notes

Saved plans are stored in this browser on this computer using localStorage.
