# Garden Planner v108 – card sizing, icons, and dark UI polish

## What changed

- Plant cards are smaller and less tall in the default browsing state.
- Default plant card now shows the big image plus the key metrics: size, water, and sun.
- Selected card expands with extra details instead of showing everything all the time.
- Removed the separate Place Plant button; clicking the card selects/activates the plant.
- Compact card footer now says More details.
- Size icon changed to a ruler.
- Water icon cleaned up, removed the weird extra line.
- Pollinator icon changed to a bee.
- Price text now wraps inside the tile instead of sticking out.
- Top canvas toolbar now uses dark-mode styling.
- Right inspector and zone settings modal now use dark-mode contrast fixes.
- Header version updated to Garden Planner ver. 108.
