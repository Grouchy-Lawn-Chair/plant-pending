# Plant Pending v1.0 — v114 UI polish

## Changes

- Moved plant search out of the Filters panel and into the Browse Plants panel above the plant list.
- Removed the close button from the plant library flyout; use the left rail icon to toggle it.
- Added a Canvas tab to the right inspector rail for display controls.
- Moved display, labels, icon opacity, merge behavior, and zoom controls into the Canvas inspector tab.
- Slimmed the top canvas toolbar so it focuses on background image, scale, drawing zones, and clear-plants actions.
- Tightened plant card icon and metric spacing.
- Kept water-level filters in the Filters panel.
