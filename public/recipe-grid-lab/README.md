# Recipe Grid Laboratory

Phase 1 development tool for Plant Pending 2.0.

Open locally at:

`http://localhost:5173/plant-pending/recipe-grid-lab/`

Depending on the Vite base path used locally, this may also be:

`http://localhost:5173/recipe-grid-lab/`

## First recipe

Elegant Privacy Hedge Border

- UpStanding Emerald Arborvitae, continuous privacy hedge
- Eau De Parfum Blush Rose, repeated flowering masses
- Early Evolution Hydrangea, interlocking middle matrix
- Bloodstone Thrift, low foreground edge

## Current controls

- Grid size, 24 × 24, 36 × 36, or 48 × 48
- Rectangle, narrow, wide, curved, or irregular test shape
- Seed 1 through 100
- Grid and center-point visibility
- Generated plant-center counts
- Manual review notes

## Phase 1 purpose

This laboratory tests whether one approved source recipe can be represented as JSON and regenerated at multiple sizes and shapes without losing its visual identity.

It is intentionally separate from the main Plant Pending planner.
