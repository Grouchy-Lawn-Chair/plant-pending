# v106 UI redesign

## What changed

- Reworked the app shell into a design-tool style layout.
- Added a narrow left utility rail for primary workspace actions.
- Converted the plant library into a docked flyout panel that can be hidden.
- Added a separate filters panel mode.
- Added a collapsible right-side properties inspector shell.
- Updated plant library cards to a more visual, metric-first layout.
- Added the version display to the header: Garden Planner ver. 106.

## Notes

This release is focused on visual hierarchy and workspace organization. It does not change the underlying garden-generation logic or database pipeline.
