# Garden Planner v107 – UI contrast + plant card refresh

## What changed

1. Improved inspector and modal contrast
   - Rebalanced many light-panel text colors from faint gray to darker slate tones.
   - Strengthened borders and field contrast in Zone Settings and inspector panels.
   - Kept the right inspector consistently light while the main workspace remains dark.

2. Refreshed plant library cards
   - Larger top image area.
   - Stronger visual hierarchy for common name and botanical name.
   - Block-based metric layout inspired by the reference card style.
   - Added consistent line-style icons for size, water, type, sun, pollinator, and price.
   - Clearer selected / active placement state.

3. Updated utility toolbar icons
   - Replaced emoji-style toolbar icons with cleaner line-style SVG icons.

4. Version label
   - App header updated to Garden Planner ver. 107.

## What to test

1. Open the app and confirm the header shows ver. 107.
2. Open the plant library and verify:
   - plant cards have larger images,
   - block metric layout shows clearly,
   - selected cards show an active placement state.
3. Open Zone Settings and confirm labels, inputs, helper text, and buttons are readable.
4. Check the right inspector sections (Zones, Planting Groups, Plant Legend, Test Log) for readability.
5. Verify the left utility toolbar icons look clean and consistent.
